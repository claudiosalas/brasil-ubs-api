# Brasil UBS API

API to search UBS near me based on Geolocation

[Data source](http://dados.gov.br/dataset/unidades-basicas-de-saude-ubs)

## Installation

Node +v8.12

NPM  +6.4.1

```bash
npm install
```

## Usage

```js
npm start
```
